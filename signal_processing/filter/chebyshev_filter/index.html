# Type I Chebyshev Filter

The gain response:

$$ G_n(\omega) = |H_n(j\omega)| = \frac{1}{\sqrt{1+\varepsilon^2T_n^2(\omega/\omega_0)}} $$

  • $\varepsilon$ - ripple factor
  • $\omega_0$ - cutoff frequency
  • $T_n$ - Chebyshev polynomial
